# Rocket Selection

## Selected Rocket

LOC Graduator model rocket kit.

## Reason for Selection

The LOC Graduator was selected because it provides a manageable beginner-level build while still allowing meaningful aerospace engineering analysis, including stability, mass properties, motor selection, and recovery planning.

## Selection Criteria

- Beginner appropriate
- Compatible with commercial motors
- Suitable for OpenRocket simulation
- Large enough to document meaningful build details
- Reasonable time commitment for a summer project
- Relevant to aerospace engineering interests
