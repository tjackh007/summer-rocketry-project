# Week 2 Build Log: Initial Simulation

## Goal

Begin learning OpenRocket and create an initial rocket simulation before the physical kit arrives.

## Work Planned

- Install OpenRocket
- Locate or create a LOC Graduator model file
- Identify key simulation outputs:
  - Center of gravity
  - Center of pressure
  - Stability margin
  - Predicted apogee
  - Maximum velocity
  - Maximum acceleration
  - Recovery deployment timing

## Notes

Simulation values will be preliminary until actual rocket mass and CG are measured after assembly.
